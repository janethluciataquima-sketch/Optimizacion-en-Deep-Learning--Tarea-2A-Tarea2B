# Optimizacion-en-Deep-Learning--Tarea-2A-Tarea2B
Ejercicios de Deep Learning: Análisis del comportamiento de la función de pérdida (loss) y la precisión (accuracy) en función del número de épocas.

📋 Contenido del Proyecto
1. 🎞️ Clasificación de Sentimiento (IMDB)
Se desarrolló un modelo de clasificación binaria para analizar reseñas de cine.

Técnicas: Regularización L2 y Dropout para combatir el sobreajuste (overfitting).

Aplicación en Nube Andina: Estos modelos nos permiten analizar el feedback de nuestros clientes sobre el confort y calidad de nuestro calzado de forma automatizada.

2. 🏠 Predicción de Precios (Regresión)
Implementación de un modelo para predecir valores continuos (basado en Boston Housing).

Innovación: Se integraron capas Dropout de 0.5 para estabilizar las curvas de validación.

Aplicación en Nube Andina: Optimización de costos de materia prima y predicción de precios de venta según la temporada de fibra de alpaca.

3. ⚖️ Equilibrio de Datos (SMOTE & ADASYN)
Réplica de los algoritmos de los artículos originales de Chawla (2002) y He (2008).

El Reto: Manejar el desbalance de clases cuando los casos de interés (ej. calzado con micro-defectos) son muy raros.

Técnica: Generación de datos sintéticos para nutrir el sistema de visión artificial de la fábrica.

🛠️ Herramientas Utilizadas
Lenguaje: Python 3.x 🐍

Frameworks: TensorFlow / Keras, Scikit-Learn.

Librerías: Imbalanced-learn (SMOTE/ADASYN), Matplotlib (Visualización).

Entorno: Google Colab.
